## Brain Tumor MRI Classification (Deep Learning)
**Multi-model MRI classifier for 4-class brain tumor detection (glioma, meningioma, pituitary, no-tumor).**

Built and compared multiple CNN architectures (baseline CNN → refined lightweight CNN → ResNet-18 from scratch → custom deep CNN) to evaluate tradeoffs between accuracy, generalization, and clinical usefulness.

**What I did**
- Dataset prep + preprocessing workflow (cropping/resizing), 70/15/15 split 
- Trained/evaluated multiple architectures and compared learning behavior and error patterns
- Used medical-relevant metrics: precision/recall/F1, confusion matrix, ROC-AUC  

**Results (highlight)**
- Dataset: 7,023 MRI images across 4 classes 
- ResNet-18 (trained from scratch): Accuracy 0.9291, AUC up to ~0.998 depending on class 

**Tech**
Python · PyTorch · torchvision · scikit-learn metrics · matplotlib
